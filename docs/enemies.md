# Enemies

## Общие правила
Враги ходят только после перемещения игрока. В свой ход враг либо атакует, если стоит рядом с игроком (Манхэттен‑1), либо двигается по своему паттерну. После атаки игрока, если цель‑враг выжила, она делает немедленную контратаку.

## Параметры
`aggroRange` — базовый радиус агрессии для большинства врагов.  
`guardRange` — радиус, в котором Guard начинает реагировать (по умолчанию равен `aggroRange`).  
`ambushRange` — дистанция, на которой Ambusher “просыпается”.  
`ambushSpeed` — сколько клеток Ambusher может пройти за ход после активации.  
`windUp` — у Bruiser: если он действовал в прошлом ходу, следующий ход пропускается.

## Типы врагов

| Тип | Поведение |
| --- | --- |
| Chaser | Преследователь. Если игрок в радиусе агро — сближается. Если рядом — атакует. |
| Bruiser | Тяжелый боец. Урон и HP выше. После действия пропускает следующий ход (wind‑up). |
| Skirmisher | Бьет и отступает на свободную клетку, увеличивая дистанцию. |
| Guard | Не идет к игроку. Если игрок подошел в радиус — атакует только вблизи. |
| Ambusher | Стоит на месте, пока игрок не приблизится. После активации быстро сближается (до `ambushSpeed` клеток). |

## Где менять логику
Основная логика поведения находится в `src/entities/enemy.js`.  
HP/ATK по типам задаются в `ENEMY_STATS`.
